# Dev Quiz Game

Repository: https://github.com/Armin-000/Quiz

Dev Quiz Game is a lightweight, skill-based quiz application for web developers.  
The quiz focuses on core technical topics such as HTTP, SQL, JavaScript, Git and general web development concepts.

The purpose of the project is to evaluate decision making and technical understanding under time pressure, rather than simple memorization.

---

## Features

- Clean and structured user interface
- Two gameplay modes
  - Normal mode (120 seconds, no penalty for wrong answers)
  - Hardcore mode (75 seconds, wrong answers reduce score)
- Questions are categorized by difficulty
  - easy, medium, hard
- Weighted scoring system
- Streak system with bonus points
- Live progress indicators
- Detailed result summary at the end of the run
- Personal best stored locally in the browser
- Fully client-side implementation (no backend)

---

## Tech Stack

- HTML  
- CSS (custom styles, no framework)  
- Vanilla JavaScript

No build tools or external dependencies are required.
